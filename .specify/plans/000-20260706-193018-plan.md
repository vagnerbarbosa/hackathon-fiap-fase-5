**Best Practices** (from Context7 Pydantic v2):
- Use `pydantic.v1` compatibility mode not needed — we're on v2
- Prefer `BaseModel` over dataclasses for validation
- Use `Field()` for constraints and defaults
- Use `typing.Literal` for string enums with fixed values
- Standard Python `Enum` for multi-value enums
- UUID validation automatic with `UUID` type

## 4. Phase 1: Design & Contracts

### 4.1 Data Model: Detection Domain

**BoundingBox** — Pixel coordinates of detected component
```python
class BoundingBox(BaseModel):
    x_min: int  # Top-left corner X
    y_min: int  # Top-left corner Y  
    x_max: int  # Bottom-right corner X
    y_max: int  # Bottom-right corner Y
```

**Point** — Center point of component
```python
class Point(BaseModel):
    x: int  # X coordinate
    y: int  # Y coordinate
```

**DetectedComponent** — Single detected architectural component
```python
class DetectedComponent(BaseModel):
    id: str                           # UUID as string
    type: str                         # Component type (user, api, database, etc.)
    confidence: float = Field(ge=0.0, le=1.0)  # YOLO confidence score
    bbox: BoundingBox                 # Bounding box coordinates
    center: Point                     # Center point for distance calculations
```

**DataFlow** — Connection between components
```python
class DataFlow(BaseModel):
    source_id: str                    # Source component UUID
    target_id: str                    # Target component UUID
    direction: Literal["unidirectional", "bidirectional"]
    inferred: bool                    # True if detected by heuristic
```

**ArchitectureGraph** — Complete detection result
```python
class ArchitectureGraph(BaseModel):
    components: list[DetectedComponent]
    data_flows: list[DataFlow]
    trust_boundaries: list[list[str]]  # Groups of component IDs by trust zone
```

### 4.2 Data Model: STRIDE Domain

**Severity** — Threat severity levels
```python
class Severity(str, Enum):
    CRITICAL = "critical"
    HIGH = "high"
    MEDIUM = "medium"
    LOW = "low"
    INFO = "info"
```

**Threat** — Single STRIDE threat identified
```python
class Threat(BaseModel):
    id: str                           # UUID as string
    category: str                     # S, T, R, I, D, or E
    component_id: str                 # Affected component UUID
    component_type: str               # Type of affected component
    description: str                  # Human-readable threat description
    severity: Severity                # Threat severity level
    affected_data_flows: list[str] = []  # Related data flow IDs
```

### 4.3 Data Model: Vulnerability Domain

**Countermeasure** — Mitigation for a threat
```python
class Countermeasure(BaseModel):
    title: str                        # Short title
    description: str                  # Detailed description
    owasp_ref: str | None = None      # Reference to OWASP cheat sheet
```

**EnrichedThreat** — Threat with vulnerability data
```python
class EnrichedThreat(BaseModel):
    id: str                           # UUID as string
    category: str                     # S, T, R, I, D, or E
    component_id: str                 # Affected component UUID
    component_type: str               # Type of affected component
    description: str                  # Human-readable threat description
    severity: Severity                # Threat severity level
    cwe_id: str | None = None         # CWE identifier (e.g., "CWE-522")
    cwe_name: str | None = None      # CWE name
    cve_ids: list[str] = []           # Related CVE identifiers
    countermeasures: list[Countermeasure] = []  # Mitigations
```

### 4.4 Data Model: Job Domain

**JobStatus** — Processing status
```python
class JobStatus(str, Enum):
    PENDING = "pending"
    PROCESSING = "processing"
    COMPLETED = "completed"
    FAILED = "failed"
```

**Job** — Analysis job tracking
```python
class Job(BaseModel):
    id: UUID = Field(default_factory=uuid4)
    status: JobStatus
    input_image_path: str
    output_report_path: str | None = None
    error_message: str | None = None
    created_at: datetime
    updated_at: datetime
```

## 6. Contracts

### Import Contract
```python
# Consumers import from domain.models
from domain.models import (
    ArchitectureGraph,
    DetectedComponent,
    DataFlow,
    Threat,
    EnrichedThreat,
    Job,
    JobStatus,
    Severity,
    BoundingBox,
    Point,
    Countermeasure,
)
```

### Validation Contract
All models validate on instantiation:
- `BoundingBox`: x_min < x_max, y_min < y_max
- `DetectedComponent`: confidence in [0.0, 1.0]
- `DataFlow`: direction is Literal value
- `Severity`: enum value
- `JobStatus`: enum value
